# voice-genius-platform

Create a new SaaS platform that helps businesses send AI generated voice notes at scale on whatsapp and telegram using voice cloning services like Eleven labs, creating an app that allows users to integrate their CRM's and connecting communication API's to be able to send the voice notes on whatsapp, etc

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/voice-genius-platform.git
cd voice-genius-platform
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
